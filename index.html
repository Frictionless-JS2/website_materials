Soft opening: Wednesday afternoons, 2-6 PM at the CATalyst Data Studio in the University of Arizona Main Library. Walk in the front entrance (north side of the building) and turn to your left.

For background on Code Commons, please see this blog post.

Code Commons is a professional and inclusive gathering that does not tolerate harassment or threatening behavior. Please treat everyone with respect and excellence, as you would be expected to at a modern workplace. People attending Code Commons are subject to the University Libraries code of conduct.
